Notes on Git and GitHub

In this page you can find some personal notes on the use of Git and GitHub. Though the notes themselves were created by myself, the content in them is the result of courses that I've taken on the subject and books I've read. This notes should not be considered as a replacement for engaging with those resources.


Some resources that I specially recommend and who's influence can be found all across these notes are:

  • "The ultimate Git course" by Mosh Hamedani's which you can find here. I personally consider this to be THE course on Git and GitHub for new comers or people who have been learning-by-doing.
  • "Happy Git with R" by Jenny Bryan which you can find here
  • Several YouTube channels that have helped me inmensly from time to time like Corey Shafer, The Net Ninja, The Modern Coder, Web Dev Simplified, GitKraken's YouTube channel.
  • The official documentation for Git and GitHub. These can be hard to navigate at first.
  • Oh Shit, Git!?! A funny guide into how to solve some of the most common mistakes we've all done when starting with Git.
